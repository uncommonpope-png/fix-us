## 🚀 Quick Start

### Install All Dependencies
```bash
cd ~/fix-us/copilot-army
pip install -r requirements.txt
```

### Initialize Army
```bash
python scripts/commander.py --init
```

### Deploy Specialists
```bash
# Deploy all copilots
python scripts/commander.py --deploy-all

# Deploy specific copilot
python scripts/commander.py --deploy code-copilot

# List all copilots
python scripts/commander.py --list
```
