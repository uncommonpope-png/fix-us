# Library Update System

**Status:** Ready for updates  
**Last Update:** March 22, 2026

---

## Purpose

This directory contains scripts and configs for updating the PLT Press library with new content, pages, and assets.

---

## Usage

Feed new content to the system. Scripts will:
1. Parse incoming data
2. Generate new pages
3. Update library index
4. Commit to GitHub
5. Deploy to GitHub Pages

---

## Files

- `scripts/` - Update automation scripts
- `config/` - Configuration files
- `docs/` - Update documentation
